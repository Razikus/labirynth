Kompilacja

Kompilacja programu to proste wywołanie skryptu ./BUILD.sh


Uruchamianie

Przechodzimy do katalogu dist, po czym uruchamiamy serwer komendą:
```
`$ java -jar LabiryntServer.jar
```
Do uruchomienia potrzebna jest sieć i znajdujący się w tym samym katalogu odpowiedni config.xml


Konfiguracja

Należy edytorem tekstu wyedytować config.xml
